# Демо-конфигурация: Криптография в 1С

Учебная конфигурация с рабочими примерами для каждого урока курса.

## Что внутри конфигурации

### Общие модули (библиотека)

| Модуль | Что делает |
|--------|-----------|
| `КриптоХеширование` | Хеширование строк, файлов, паролей с солью |
| `КриптоШифрование` | Шифрование и расшифровка данных |
| `КриптоПодпись` | Подписание и проверка ЭЦП |
| `КриптоHTTPS` | HTTPS-запросы, клиентские сертификаты |

### Обработки-демо

| Обработка | Урок | Что демонстрирует |
|-----------|------|-------------------|
| `Урок02_Хеширование` | 02 | Хеш строки, файла, изменение хеша |
| `Урок03_Пароли` | 03 | Хранение пароля с солью, проверка |
| `Урок05_Шифрование` | 05 | Шифрование и расшифровка строки |
| `Урок06_Подпись` | 06 | Подписание файла, проверка подписи |
| `Урок07_РеальныеЗадачи` | 07 | HTTPS-запрос, JWT |

## Установка

### Вариант 1: Готовый .cf файл (быстро)

1. Скачайте файл из папки `releases/crypto-1c-demo.cf`
2. Создайте новую пустую базу 1С
3. Конфигуратор → Конфигурация → Загрузить конфигурацию из файла
4. Выберите `crypto-1c-demo.cf`
5. Запустите базу в режиме предприятия

### Вариант 2: Из исходников EDT (для изучения)

1. Установите 1C:EDT или используйте конфигуратор с поддержкой формата EDT
2. Откройте папку `src/` как проект

## Требования

- Платформа 1С:Предприятие **8.3.19** и выше
- Windows (для примеров с Microsoft CSP)
- КриптоПро CSP — **не обязателен** для базовых примеров

## Как использовать код из конфигурации в своём проекте

Общие модули (`КриптоХеширование`, `КриптоШифрование` и др.)
спроектированы как независимые библиотеки. Скопируйте нужный модуль
в свою конфигурацию — он будет работать без изменений.

Зависимости между модулями: `КриптоПодпись` использует `КриптоХеширование`.
Остальные модули независимы друг от друга.
