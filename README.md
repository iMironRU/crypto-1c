# Криптография в 1С: практический курс

Учебный курс для разработчиков 1С, которые знают язык,
но никогда не работали с криптографией.

## Что внутри

**8 уроков** — от «зачем это вообще нужно» до «как диагностировать ошибки в продакшне».
Каждый урок рассчитан на 45 минут и содержит теорию, рабочий код и контрольные вопросы.

**Библиотека кода** — готовые функции для хеширования, шифрования,
электронной подписи и HTTPS-запросов. Берёте файл, вставляете в проект, используете.

**Демо-конфигурация** — обработки 1С для каждого урока.
Запускаете, видите результат, изучаете код прямо в конфигураторе.

## Что нужно для начала

- Платформа 1С:Предприятие 8.3.19 или новее
- Windows (для встроенного криптопровайдера)
- Конфигуратор для работы с демо-конфигурацией

КриптоПро и токены **не нужны** для прохождения курса.
Все примеры работают на встроенных средствах Windows.
Там где в реальной работе нужен ГОСТ — это явно указано.

## Как пользоваться

**Если хотите пройти курс** — читайте уроки по порядку,
начиная с [Урока 01](course/lesson-01/README.md).
После каждого урока — контрольные вопросы, не пропускайте.

**Если нужен конкретный код** — смотрите папку [/code](code/).
Каждый файл самодостаточен и содержит описание в комментариях.

**Если хотите потрогать руками** — установите
[демо-конфигурацию](configuration/README.md) и запускайте обработки.
